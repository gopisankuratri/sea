# SEA Farms Welcome Page

Minimal React + TypeScript + Vite app for the SEA Farms web application.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Deploy on Netlify

Use these settings:

```text
Build command: npm run build
Publish directory: dist
```

This version contains only the welcome page. Google Sheets and ChatGPT integration will be added in the next phase.
